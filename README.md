# Dev Platform project
-----------
## Project Description
> 내부 컨테이너로 동작하는 VSCode 컨테이너를 제공하여, 개발 플랫폼을 만드는 프로젝트 입니다.
> 내부 컨테이너에서, 알고리즘, 인공지능 Code Template 등을 제공하고, 커스터마이징 기능을 제공합니다.

### Gateway
> Spring cloud gateway 로 구성된 API-Gateway 입니다.

## Language & Build tool
+ Java & Gradle

## Support
+ Kubernetes

+ Java 8
